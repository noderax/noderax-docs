# API Overview

The Noderax API is a NestJS control-plane API exposed under `/api/v1` for most
application routes, with selected health routes intentionally exposed without
the prefix.

## Documentation model

This section combines:

- hand-authored operational guides
- generated endpoint reference pages derived from a checked-in OpenAPI schema snapshot

## Why the schema is checked in

The docs site is a separate repo and should not depend on a live production
Swagger endpoint. Instead:

- `noderax-api` exports a deterministic `openapi.json`
- `noderax-docs` keeps a local schema snapshot under `openapi/openapi.json`
- the generated endpoint tree is produced from that local file

<Callout type="info">
  v1 intentionally avoids a live authenticated playground or cookie-forwarding
  proxy. The generated reference is read-only by design.
</Callout>
