# Recent Platform Changes

Recent platform work hardened several sensitive areas:

- installer-first setup and guided runtime promotion
- domain-aware TLS and cert issuance validation
- control-plane self-update with prepared/apply semantics
- split Updates Center for control plane versus agent updates
- full-screen freeze modal during control-plane apply
- corrected runtime health and route probing
- HA-safe realtime fanout for node install progress and live metrics

These changes are already reflected throughout the docs tree. This page exists
to provide a high-level release narrative for the current control-plane line.
